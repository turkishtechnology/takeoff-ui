import { TkBadge } from "@takeoff-ui/react"; 

### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| <TkBadge label="containerStyle" variant="primary" size="large" type="filledlight"/> | <code>any</code> | null | The style attribute of container element |
| <TkBadge label="description" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Sets description text for the tooltip. |
| <TkBadge label="header" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Sets header text for the tooltip. |
| <TkBadge label="icon" variant="primary" size="large" type="filledlight"/> | <code>IIconOptions, string</code> | null | Sets the icon element of the tooltip. |
| <TkBadge label="position" variant="primary" size="large" type="filledlight"/> | <code>"bottom", "bottom-end", "bottom-start", "left", "left-end", "left-start", "right", "right-end", "right-start", "top", "top-end", "top-start"</code> | 'right' | Sets the position of the tooltip. |
| <TkBadge label="variant" variant="primary" size="large" type="filledlight"/> | <code>"danger", "dark", "info", "neutral", "success", "warning", "white"</code> | 'neutral' | Sets the color variant of the tooltip. |

### Slots

| Name | Description |
| ---- | ----------- |
| content | Define custom HTML content for the tooltip, which replaces the default header, description and icon elements |
| trigger | The trigger slot defines the element that will trigger the tooltip |
