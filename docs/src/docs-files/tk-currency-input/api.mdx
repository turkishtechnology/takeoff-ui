import { TkBadge } from "@takeoff-ui/react"; 

### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| <TkBadge label="currency" variant="primary" size="large" type="filledlight"/> | <code>string</code> | 'TL' | Specifies the currency to be displayed. |
| <TkBadge label="currencyOptions" variant="primary" size="large" type="filledlight"/> | <code>any[]</code> | null | Defines the available currency options for the user to select from. |
| <TkBadge label="decimalSeparator" variant="primary" size="large" type="filledlight"/> | <code>string</code> | ', ' | Sets the character used to separate the decimal part. |
| <TkBadge label="disabled" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | If `true`, the user cannot interact with the input. |
| <TkBadge label="error" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | This is the error message that will be displayed. |
| <TkBadge label="hint" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Provided a hint or additional information about the input. |
| <TkBadge label="icon" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Specifies a material icon name to be displayed. |
| <TkBadge label="invalid" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Indicates whether the input is in an invalid state. |
| <TkBadge label="label" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Defines the label for the input. |
| <TkBadge label="name" variant="primary" size="large" type="filledlight"/> | <code>string</code> | this.inputId | The name of the control, which is submitted with the form data. |
| <TkBadge label="placeholder" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Placeholder text displayed when the input is empty. |
| <TkBadge label="precision" variant="primary" size="large" type="filledlight"/> | <code>number</code> | 2 | Determines the number of decimal places to display, like 2 decimal places for cents. |
| <TkBadge label="readonly" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | If `true`, the user cannot modify the value. |
| <TkBadge label="size" variant="primary" size="large" type="filledlight"/> | <code>"base", "large", "small"</code> | 'base' | Sets size for the component. |
| <TkBadge label="thousandsSeparator" variant="primary" size="large" type="filledlight"/> | <code>string</code> | '.' | Specifies the character used to separate thousands in large numbers. |
| <TkBadge label="value" variant="primary" size="large" type="filledlight"/> | <code>number</code> | 0 | The value of the input. |

### Events

| Name | Description |
| ---- | ----------- |
| tk-blur | Emitted when the input loses focus. |
| tk-change | Emitted when the value has changed. |
| tk-focus | Emitted when the input has focus. |
| tk-input | Emitted when a keyboard input occurred. |

### Methods

| Name | Description |
| ---- | ----------- |
| setFocus | Sets focus on the specified `tk-input`. Use this method instead of the global  `input.focus()`. |
