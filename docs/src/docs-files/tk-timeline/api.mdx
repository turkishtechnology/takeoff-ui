### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| <TkBadge label="alternate" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | true | Whether to alternate the position of timeline items relative to the line. |
| <TkBadge label="items" variant="primary" size="large" type="filledlight"/> | <code>TimelineItem[]</code> | [] | An array of objects representing the items to display on the timeline.  Each object should have at least a `title`. `description` and `date` are optional. |
| <TkBadge label="orientation" variant="primary" size="large" type="filledlight"/> | <code>"horizontal", "vertical"</code> | 'horizontal' | The orientation of the timeline. |

