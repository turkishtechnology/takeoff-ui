import { TkBadge } from "@takeoff-ui/react"; 

### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| <TkBadge label="disabled" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | If true, disables all interaction with the tree view. |
| <TkBadge label="mode" variant="primary" size="large" type="filledlight"/> | <code>"basic", "stepper"</code> | 'basic' | Tree view mode: 'basic' or 'stepper'. |
| <TkBadge label="showBadge" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | true | Show/hide the badge for children count on directories. |
| <TkBadge label="showExpandIcon" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | true | Show/hide the directory arrow icon. |
| <TkBadge label="showFileIcon" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | true | Show/hide the file icon for files. |
| <TkBadge label="showFolderIcon" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | true | Show/hide the folder icon for directories. |
| <TkBadge label="size" variant="primary" size="large" type="filledlight"/> | <code>"base", "large", "small"</code> | 'base' | Tree view size: 'base', 'small', or 'large'. |
| <TkBadge label="type" variant="primary" size="large" type="filledlight"/> | <code>"basic", "divided", "light"</code> | 'basic' | Tree view type: 'basic', 'divided', or 'light'. |

### Events

| Name | Description |
| ---- | ----------- |
| tk-item-click | Event emitted when a tree item is clicked. |
