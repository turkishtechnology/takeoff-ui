## Mode

The `mode` prop controls the display mode of the tree view. It can be set to `basic` or `stepper`.

<TreeViewMode />

## Type

The `type` prop controls the style type of the tree view. It can be set to `basic`, `divided`, or `light`.

<TreeViewType />

## Size

The `size` prop controls the size of the tree view. It can be set to `base`, `small`, or `large`.

<TreeViewSize />

## Icons

The `showExpandIcon`, `showFolderIcon`, `showFileIcon`, and `showBadge` props control the visibility of the expand icon, folder icon, file icon, and badge.

<TreeViewIcons />

## Disabled

The `disabled` prop controls the disabled state of the tree view.

<TreeViewDisabled />
