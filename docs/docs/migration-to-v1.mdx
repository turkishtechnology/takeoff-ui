## Breaking Changes 

### Changes of Table Component

* <code>paginationType</code> has been changed.

* Now there are <code>paginationType</code> and <code>paginationMethod</code> properties.

- paginationMethod: 
    - Defines whether pagination is handled on the `client` or `server` side.

+ paginationType: 
    + Defines the design of the pagination which can be either `grouped`, `outlined` or `text`.

### Changes of Upload Component

* tkFilesAccepted emit is not supported anymore.

* Now file changes are managed by `tkChange` emit.
