import { TkBadge } from "@takeoff-ui/react"; 

### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| <TkBadge label="ariaLabelledby" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | The aria-labelledby attribute of the toggle. |
| <TkBadge label="disabled" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Whether the toggle is disabled. |
| <TkBadge label="icon" variant="primary" size="large" type="filledlight"/> | <code>string</code> | 'check' | Specifies a material icon name to be displayed. |
| <TkBadge label="inputId" variant="primary" size="large" type="filledlight"/> | <code>string</code> | '' | The ID of the input element. |
| <TkBadge label="invalid" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Whether the toggle is in an invalid state. |
| <TkBadge label="label" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | The label for the toggle. |
| <TkBadge label="name" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | The name attribute of the toggle. |
| <TkBadge label="showIcon" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | true | Whether to show the icon in the toggle. |
| <TkBadge label="size" variant="primary" size="large" type="filledlight"/> | <code>"base", "large", "small", "xlarge", "xsmall"</code> | 'base' | Sets size for the component. |
| <TkBadge label="value" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | The current state of the toggle. |
| <TkBadge label="variant" variant="primary" size="large" type="filledlight"/> | <code>"info", "success"</code> | 'info' | The type of the toggle. |

### Events

| Name | Description |
| ---- | ----------- |
| tk-change | Event emitted when the toggle value changes. |

### Methods

| Name | Description |
| ---- | ----------- |
| getInputElement | Returns the native `<input>` element used under the hood. |
