import { TkBadge } from "@takeoff-ui/react"; 

### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| <TkBadge label="checked" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Marks the radio button as checked or unchecked. |
| <TkBadge label="description" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | The description sub text displayed. |
| <TkBadge label="disabled" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Disables the radio button if true. |
| <TkBadge label="invalid" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Indicates whether the input is in an invalid state |
| <TkBadge label="label" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Defines the label for the element. |
| <TkBadge label="name" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | The name of the radio group, used to group radio buttons together. |
| <TkBadge label="position" variant="primary" size="large" type="filledlight"/> | <code>"left", "right"</code> | null | Determines the position of the radio and label. |
| <TkBadge label="value" variant="primary" size="large" type="filledlight"/> | <code>any</code> | null | The value of the radio button. |

### Events

| Name | Description |
| ---- | ----------- |
| tk-change | Emitted when the radio button's checked state changes. |

### Slots

| Name | Description |
| ---- | ----------- |
| content | Custom content template. |
