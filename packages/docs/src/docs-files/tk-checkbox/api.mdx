import { TkBadge } from "@takeoff-ui/react"; 

### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| <TkBadge label="description" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | The description sub text displayed. |
| <TkBadge label="disabled" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | If true, the user cannot interact with the checkbox. |
| <TkBadge label="indeterminate" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | If true, the checkbox will be indeterminate. |
| <TkBadge label="invalid" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Indicates whether the input is in an invalid state |
| <TkBadge label="label" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Defines the label for the checkbox. |
| <TkBadge label="name" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Name of the checkbox |
| <TkBadge label="type" variant="primary" size="large" type="filledlight"/> | <code>"card", "default"</code> | 'default' | Determines the appearance types of the checkbox. |
| <TkBadge label="value" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Sets the checkbox value |

### Events

| Name | Description |
| ---- | ----------- |
| tk-change | Emitted when the checkbox checked state changes. |
