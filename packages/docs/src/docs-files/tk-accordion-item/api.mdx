import { TkBadge } from "@takeoff-ui/react"; 

### Props

| Name | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| <TkBadge label="active" variant="primary" size="large" type="filledlight"/> | <code>boolean</code> | false | Sets if the accordion is active. |
| <TkBadge label="header" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Header text to display. |
| <TkBadge label="icon" variant="primary" size="large" type="filledlight"/> | <code>IIconOptions, string</code> | null | Icon for accordion component. |
| <TkBadge label="itemKey" variant="primary" size="large" type="filledlight"/> | <code>string</code> | null | Optional key for the accordion item. |
| <TkBadge label="size" variant="primary" size="large" type="filledlight"/> | <code>"base", "large"</code> | 'base' | Sets size for the component. |
| <TkBadge label="toggleItem" variant="primary" size="large" type="filledlight"/> | <code>()=>void</code> | null | Toggle's the accordion item. |

### Slots

| Name | Description |
| ---- | ----------- |
| content | Custom content template. |
| header | Custom header template that overrides the header prop if provided. |
